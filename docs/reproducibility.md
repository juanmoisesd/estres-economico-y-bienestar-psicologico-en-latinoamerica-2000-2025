# Reproducibility Guide

## Dataset: Dataset longitudinal (2000-2025) que examina la relación entre indicadores de estrés económico y salud mental en 21 países de América Latina e Iberia. Variables: PIB per cápita, tasa de desempleo, inflación, coeficiente Gini, tasa de pobreza, índice 

---

## 1. Requirements

- Web browser or data analysis software (R, Python, Excel)
- No special software required for basic data access

## 2. Data Access

```
# Clone repository
git clone https://github.com/juanmoisesd/estres-economico-y-bienestar-psicologico-en-latinoamerica-2000-2025.git
cd estres-economico-y-bienestar-psicologico-en-latinoamerica-2000-2025
```

## 3. Data Format

- Data is provided in standard CSV/HTML format
- UTF-8 encoding
- See `data_dictionary.csv` or README for variable descriptions
